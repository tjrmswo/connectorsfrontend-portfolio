## 설치 및 실행
```bash
# 클론 및 설치
git clone https://github.com/goldencompass2022/connectorsfrontend
cd connectorsfrontend
pnpm install

# 개발 서버 실행 (HTTPS)
pnpm devs

# Docker로 실행
docker build -t connectors .
docker run -d -p 3000:3000 --name connectingRoad connectors
```
